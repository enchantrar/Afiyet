# 🍽️ Afiyet

> *Afiyet olsun*, a Turkish wish of good health through food.

**Afiyet** is an AI-powered menu generator that crafts full 3-course menus based on your chosen cuisine. It includes a dedicated medical diet mode for patients managing conditions like GI cancer, diabetes, heart disease, gout, and more.

## Features

- 🌍 **10 cuisines** - Turkish, Italian, Japanese, Mexican, Indian, French, Chinese, American, Thai, Greek
- 🍽️ **General mode** - dietary preference filters (vegetarian, vegan, keto, coeliac, gluten-free, lactose-free...) + spice level
- 🩺 **Medical diet mode**, condition-specific strict dietary rules:
  - 🎗️ GI Cancer / Chemotherapy
  - 🩸 Hypoglycaemia
  - 💉 Type 2 Diabetes
  - ❤️ Heart / Cardiovascular
  - 🦴 Gout (Low Purine)
  - 🫁 Tuberculosis Recovery
- 🛒 **Shopping list**, optional ingredient list with every menu
- 🎲 **Surprise Me**, random cuisine picker
- 🌙 **Dark mode**, full light/dark theme toggle
- 🇬🇧 / 🇹🇷 **English & Turkish**, full UI and menu output in both languages


## Tech

Single-file HTML/CSS/JavaScript app. No build tools, no dependencies, no backend.  
Menu generation powered by Gemini API.

## Project Management

Tracked on Notion task board with phases: Web MVP → Features → Backend → Mobile.

## Origin

The medical diet feature was inspired by personal experience caring for a family member with gastrointestinal cancer. Afiyet is built with the hope that it helps patients and their families find nourishing, safe meals during difficult times. 💙
